# 🏨 Baringo Wonders Safari Booking System

A luxurious, modern booking system for Baringo Wonders Safari with Kenyan payment integration and professional UI design.

## ✨ Features

### 🎯 Core Functionality
- **Multi-step Booking Flow**: Service → Date/Time → Customer Details → Payment → Confirmation
- **Kenyan Currency**: All prices displayed in Kenyan Shillings (KES)
- **Multiple Payment Methods**: M-Pesa, PayPal, Visa Card, Google Pay
- **Professional Glass UI**: Modern glassmorphism design with animations
- **Print Receipts**: Generate professional booking receipts
- **Responsive Design**: Works perfectly on all devices

### 💳 Payment Integration
- **M-Pesa**: Kenya's leading mobile money service
- **PayPal**: International payment option
- **Visa Card**: Credit/Debit card processing
- **Google Pay**: Modern digital wallet

### 🎨 Design Features
- **Glass Cards**: Beautiful backdrop blur effects
- **Lake Theme**: Serene blue gradient backgrounds
- **Animations**: Smooth transitions and hover effects
- **Professional Typography**: Clean, modern font hierarchy
- **Color-coded States**: Visual feedback for form completion

## 🚀 Getting Started

### Prerequisites
- Node.js (version 14 or higher)
- npm or yarn package manager

### Installation

1. **Navigate to the project directory:**
   ```bash
   cd "C:\Users\HP\Desktop\LAKE BARINGO HOTEL"
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm start
   ```

4. **Open your browser and navigate to:**
   ```
   http://localhost:3000
   ```

## 🔄 Booking Process

1. **Select Service**: Choose from available hotel experiences
2. **Choose Date & Time**: Pick preferred schedule
3. **Customer Information**: Fill in personal details
4. **Payment**: Select and complete payment method
5. **Confirmation**: Receive booking confirmation and receipt

## 🖨️ Print Receipt Feature

After successful booking, customers can:
- Click "Print Receipt" button
- Generate professional PDF receipt
- Includes all booking and payment details
