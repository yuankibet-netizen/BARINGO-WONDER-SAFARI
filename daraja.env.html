Sure, I can guide you on how to generate an access token using your consumer key and consumer secret. Here's a simple Python code snippet:

Daraja Console
import requests
import base64

consumer_key = 'your_consumer_key'
consumer_secret = 'your_consumer_secret'
api_url = 'https://sandbox.safaricom.co.ke/oauth/v1/generate?grant_type=client_credentials'

auth = base64.b64encode(f'{consumer_key}:{consumer_secret}'.encode()).decode()
headers = {'Authorization': f'Basic {auth}'}

response = requests.get(api_url, headers=headers)

if response.status_code == 200:
    access_token = response.json()['access_token']
    print(f'Access token: {access_token}')
else:
    print(f'Error: {response.json()["error_description"]}')

A short code is a unique number allocated to a pay bill or buy goods organization through which they can receive customer payments. It could be a Pay bill, Buy Goods or Till Number. If you don't have a short code, you can apply for one through the M-PESA for Business self-service portal: https://m-pesaforbusiness.co.ke/apply. Once you have a short code, you can integrate it with the Daraja API to receive payments from M-PESA customers.
